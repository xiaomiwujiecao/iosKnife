## 手机上的工具

###  `Cydia`

越狱后自动安装
 

### `iFile`

在手机上管理文件

### `AFC2` 

通过USB管理手机文件

### `MTerminal`

手机中执行命令行


### `Cycript`

代码注入测试工具

### `LLDB`和 `debugserver`

远程调试工具

### `usbmuxd` 

通过USB连接 SSH 


### `IDA` 、`Hopper`

反编译工具

### 其他工具

- `syslogd` 记录日志

- `Vi IMproved` 编辑器

- `apt` 下载命令

- `Network commands` 网络命令

- `dumpedecrypted` 脱壳

- `class-dump` 导出头文件

- `lsof` 查看进程所占用的文件

- `AppSync` 安装未签名的应用

- `App Admin` 下载指定版本的应用
